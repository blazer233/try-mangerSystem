
前后端资金系统管理
（使用时，配置server.js连接到本地数据库）

前端： vuecli3 + element-ui + axios + vue-router


后端： node+express


数据库： mongodb mongoose操纵数据库模型改变数据库中数据


登陆校验：由  sessionStorage 密码保存 



token：进行jsonwebtoken编译，设置过期时间


concurrently进行前后端连载


前端提供路由+页码


后端提供api接口连接数据库

安装环境：


cnpm i 


cd client


cnpm i


运行：


npm run dev
